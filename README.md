# last-ep-scanner
Analyse le dossier spécifié et affiche les derniers episodes des séries données.


## Config

Spécifier le chemin absolu du dossier contenant les séries, et modifier le nom du fichier qui sera créé pour stocker les séries à suivre.

L'arborescence doit être tel quel :

- Séries
    - Série1
    - Série2
        - Season 1
        - Season 2
            - Episode (ex. "Titre 1...", "Titre 2x01..."
